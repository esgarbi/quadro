# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Quadro, please report it
responsibly by opening a **private security advisory** at
https://github.com/esgarbi/quadro/security/advisories/new

Do **not** open a public GitHub issue for security vulnerabilities.

We will acknowledge your report within 48 hours and aim to provide a
fix or mitigation within 7 days.
